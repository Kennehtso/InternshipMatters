# Use heroku cli
# Windows 
1. open cmd in windows
2. check version to make sure is installed 
	- heroku --version
3. login
	- heroku login
4. popup a browser login request, input creditials to login
5. Close browser and go back to cli
6. Command
	- type 'cd {project directory clone from git}' to set our current location
	- type 'heroku git remote -v' to check there should list out our project with 'origin' and 'heroku'
	- type 'heroku git:remote -a {App name}' to access to our app
	- type 'heroku run python manage.py migrate'
